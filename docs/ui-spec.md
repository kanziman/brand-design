## S-00. 로그인 게이트 `NEW v2`

**목적**: 진입 시 로컬-우선/동기화를 담백하게 선택하게 한다. 로그인을 강제하지 않는다.
**진입**: 인증 상태가 없을 때 최초 화면.

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-00-1 워드마크 | Outfit "Momento" | — |
| S-00-2 안내 | "로그인하면 어느 기기에서든 이어서 정리할 수 있어요." | — |
| S-00-3 Google 버튼 | 중립 화이트 버튼 + 'G' 마크 | 인증 → 앱 진입(google 모드) |
| S-00-4 구분선 | "또는" | — |
| S-00-5 계정 없이 시작 | `Button variant="ghost"` | 로컬 모드로 앱 진입 |
| S-00-6 보조 안내 | "계정 없이도 로컬에 저장돼요. 로그인은 기기 간 동기화를 위한 선택이에요." | — |

- ⚠️ 'G' 마크는 임시 플레이스홀더 → 공식 Google 브랜드 SVG 교체 필요.

## G-01. 사이드바 (공통)

**목적**: 전역 내비게이션 + 프로젝트 관리 + 빠른 메모 진입 + 계정 상태.

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| G-01-1 로고 | Momento 워드마크 | 클릭 시 오늘(S-01) 이동 |
| G-01-2 내비 | 오늘 / 메모함(개수) / 전체 보기 | 현재 위치 활성(테라코타 텍스트 + `--bg-surface-raised`) |
| G-01-3 프로젝트 목록 | 색 점 + 이름 + 남은 태스크 수 | 클릭 시 S-03 이동 |
| G-01-4 새 프로젝트 | '+ 새 프로젝트'(점선 버튼) | 즉시 생성 후 S-03 진입 |
| G-01-5 빠른 메모 버튼 | '+ 빠른 메모 ⌘N' | S-05 팝오버 오픈 |
| G-01-6 계정 행 (AccountRow) `NEW v2` | 로그인: 아바타+이메일+로그아웃 / 로컬: 세이지 점 + "로컬에 저장 중 · 로그인" | 로그아웃/로그인 전환 |

- **반응형 `NEW v2`**: `<760px`에서 사이드바 숨김 → 상단 고정 바(햄버거 + 워드마크, height 56px) + 좌측 오프캔버스 드로어(슬라이드, 스크림). 나머지 화면은 상단 바 아래로.
- **상태**: 프로젝트 0개 → 목록 대신 새 프로젝트 안내.

## S-01. 오늘 · 지금 하나

**목적**: 모든 프로젝트를 가로질러 **지금 할 단 하나**에 몰입하게 한다.

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-01-1 헤더 | 날짜(동적) + "오늘도 한 걸음." | — |
| S-01-2 주간 요약 | "이번 주 N개 끝냈어요" | 완료만 표기(압박 수치 아님) |
| S-01-3 지금 하나 카드 `NEW v2` | **테라코타 보더(1.5px) + `--accent-soft` 배경 + "지금 하나" 라벨** + 프로젝트 태그 + 태스크명(2xl) + 예상 소요 | 하이라이트 |
| S-01-4 끝냈어요 | 지금 하나 완료 | 완료 → 다음 태스크 승격, 주간 완료 +1 |
| S-01-5 다음 하나 | 지금 것을 뒤로 미룸 | `today_seq` 뒤로, 다음 것 승격 |
| S-01-6 이어서 목록 `NEW v2` | 오늘의 나머지: 체크박스 + 제목 + 이월 라벨 + 예상 소요 | 체크=완료 · **제목 클릭 → S-07 상세** |
| S-01-7 오늘 완료 `NEW v2` | "오늘 완료 · N" 접힘 목록 | 체크 해제로 되돌리기 |

**'지금 하나' 선정 규칙**: `status=today` & 미완료 중 `today_seq` 최솟값(파생).

**상태 `NEW v2`**
- **기본**: 지금 하나 카드 노출.
- **모두 완료**: "오늘 할 일을 다 끝냈어요." 카드(+ 오늘 완료 목록).
- **빈 상태**: "오늘 할 일이 없어요…" + [전체 보기 →]·[메모함 →] 바로가기.
- **이월 항목**: '내일로 미룸' 담백한 pill. 붉은색·연체 표시 없음.

## S-02. 전체 보기

**목적**: 여러 프로젝트를 한눈에 조망하되, 각 프로젝트의 **다음 하나만** 보여 압도감을 줄인다.

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-02-1 헤더 | "N개의 프로젝트" | — |
| S-02-2 프로젝트 카드 | 색 점 + 이름 | 클릭 시 S-03, hover 시 `--shadow-card-hover` |
| S-02-3 다음 하나 | 시간순 다음 태스크(오늘 우선) | 라벨 '다음 하나' |
| S-02-4 진행 바 | 단색 바 + "N개 남음" | %가 아닌 남은 개수 |

**상태**: 다음 하나 없음 → "예정된 일 없음". 프로젝트 0개 → 온보딩 유도.
**반응형 `NEW v2`**: `<760px`에서 그리드 2열 → 1열.

## S-03. 프로젝트 상세

**목적**: 한 프로젝트의 태스크를 관리(추가·완료·오늘로 올리기).

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-03-1 헤더 | 색 점 + 프로젝트명 + '편집' | — |
| S-03-2 태스크 추가 | 한 줄 입력 "+ 할 일 추가 · Enter" | Enter로 추가 |
| S-03-3 태스크 목록 | 체크박스 + 제목 + 상태/예상 소요 | 체크=완료 · **제목 클릭 → S-07 상세** |
| S-03-4 오늘로 `NEW v2` | 미완료 & 오늘 아님 → '오늘로 ↑' pill(`--accent-soft`) / 오늘이면 '오늘' 라벨 | `status=today` + `today_seq` 부여 |
| S-03-5 완료 섹션 | "완료 · N" 목록 | 되돌리기 |

**상태**: 빈 상태 → "첫 할 일을 적어 보세요."

| S-04-4 삭제 | 메모 제거 | — |

**상태**: 빈 상태 → "비어 있어요. 떠오른 생각은 ⌘N으로 담아두세요."

## S-05. 빠른 메모 팝오버

**목적**: 몰입을 끊지 않고 떠오른 생각을 한 줄로 담는다.
**진입**: ⌘N(입력 필드 밖) / 사이드바 버튼.

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-05-1 라벨 `NEW v2` | "빠른 메모"(소형 캡션) | — |
| S-05-2 입력창 | 한 줄, 자동 포커스, "떠오른 생각을 한 줄로…" | 타이핑 |
| S-05-3 안내/단축키 `NEW v2` | "메모함에 담고 하던 일로 돌아가요" · `Enter` 저장 · `Esc` 닫기 | — |
| S-05-4 저장 | Enter | 메모함에 추가 후 닫힘 → 작업 복귀 |
| S-05-5 닫기 | Esc / 바깥 클릭 | 저장 없이 닫힘 |

- 중앙 팝오버, 스크림 `rgba(26,26,26,0.15)`(블러 없음).

## S-06. 온보딩 · 빈 상태

**목적**: 최초 실행 시 첫 프로젝트·첫 태스크로 자연스럽게 유도.

| 단계 | 내용 |
|------|------|
| 1 | 간단 소개 1문장 + '첫 프로젝트 만들기' |
| 2 | 프로젝트 이름·색 선택 |
| 3 | 첫 태스크 3개 입력 → 하나를 '오늘'로 |
| 완료 | S-01(오늘) 진입, 지금 하나 노출 |

- `NEW v2` — 하이파이 프로토타입은 **S-00 로그인 게이트**를 진입점으로 두고, '계정 없이 시작하기' 후 시드/빈 상태에서 시작한다. 3단계 온보딩 위저드는 v1 UI 킷 정의이며, 로그인 게이트와 함께 쓰거나 대체 가능(구현 시 택1).

**진입**: 태스크 제목 클릭. **컨테이너**: 우측 슬라이드오버(폭 `min(440px,94vw)`, `--shadow-float`, 스크림 블러 없음).

| 컴포넌트 | 설명 | 동작 |
|----------|------|------|
| S-07-1 헤더 | "할 일" 라벨 + 닫기 ✕ | Esc·바깥·닫기로 종료 |
| S-07-2 제목 | 체크박스 + 제목 입력(lg) | 완료 토글 / 제목 편집 |
| S-07-3 프로젝트 | 라벨 76px + 프로젝트 태그 | — |

## 공통 규칙 요약

- **색/포인트**: 테라코타는 '지금 하나'·'오늘로' 등 핵심 신호에만. 나머지는 잉크·웜그레이.
- **압박 신호 금지**: 붉은 배지, 연체 카운트, 달성률 % 미사용. 마감일도 지난 날짜에 색 불변.
- **문구**: 담백·사실 위주. 감성 위로 과다 금지.
- **오버레이 2종**: 중앙 팝오버(빠른 메모) · 우측 슬라이드오버(할 일 상세). 스크림 동일, 블러 없음.
- **접근성**: 명도 대비, 키보드(⌘N·Enter·Esc), 포커스 표시.
- **반응형**: 데스크톱 우선, `<760px` 사이드바 오프캔버스 + 그리드 1열.
